This site was built as a tool for calculus students who are beginning to explore integrals. To learn more about integrals, check out the "Learn More" page.

  1. Select a function.
  2. Fill in values for the constants or use the preset values.
  3. Select the domain for your function.
    4. Be careful! The natural log function is undefined when x <= 0.
  4. Pick your sum type.
  5. Decide whether you want to be able to pick the number of rectangles in the sum or the width of each rectangle.
    6. This option will not be available if your sum type is Simpson's Rule.
  6. Press the graph it button.
    7. This will only work if all of your inputs are valid.

That's it!
